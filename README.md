

The original plugin was made by Mizarc. for all the info about their plugin go to the waystonewarps plugin page: https://github.com/mizarc/waystone-warps

I added Bedrock support for geyser servers.
bedrock players get a special naming ui instead of the anvil ui.

I added a co-owner menu (emerald icon). This menu allow you to give other players permission to edit your waystone (except breakking it and adding co-owners).

## new Permissions

| Permission Node                          | Description |
|------------------------------------------|-------------|
| `waystonewarps.assign_group`         | Allows player to change the group of a waystone|

## new commands

| commands                  | Description |
|-----------------------------------|-------------|
| `/warpdefaultgroup none or groupname`        | sets the default group. every new waystone wil be assigned to this group. |
